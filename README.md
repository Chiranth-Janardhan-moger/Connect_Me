# 🚌 Bus Tracking System 

## 📋 Prerequisites

- **Node.js** (v18 or higher)
- **MySQL** (v8.0 or higher)
- **Expo CLI** (`npm install -g expo-cli`)
- **Git**

## 🚀 Quick Start

### 1. Clone Repository

```bash
git clone https://github.com/Chiranth-Janardhan-moger/Connect_Me.git
cd Connect_Me
```

---

## 🔧 Backend Setup

### Step 1: Install Dependencies

```bash
cd backend
npm install
```


## 📱 Mobile App Setup

### Step 1: Install Dependencies

```bash
cd frontend
npm install
npx expo start
`
